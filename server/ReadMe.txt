//Adex Ocean Vision Sever

1). add dependencies
   npm install

2). database migration 
  npx sequelize-cli db:migrate || npm run db:migrate

3). database seed migration
  npx sequelize-cli db:seed:all || npm run db:seed
  
4). server start
  npm run start 

Prerequisites:
 1) Node(^14)
 2) OS(Linux,Mac)
